Introduction

A group project to study on object-oriented, in which users can learn on flood safety regulations through a quick quiz game, no matter the age. Created within 3-4 months and used free assets, I was tasked as the user interface designer and decided on a simple interface layout for users to navigate their way through the system.


Platform

Android, Windows (debugging)

Engine

Unity2D

Genre

Educational

Description

  • For now, the system only has a workable quiz game as this feature is the main priority.
  • A class that contains a list of questions for the quiz is randomized by a randomizer class then sends over to the UI handler class.
  • Each situational question answered will provide tips and advices, then recorded those information into a handbook to be referred in the future.
